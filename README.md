# Microblogging Web Application

A Twitter-like microblogging platform built with FastAPI (backend) and React (frontend).

## Features
- User profile creation
- Login/authentication
- Post short text updates (max 280 chars)
- Global chronological feed
- Like posts
- Reply to posts (one level deep)
- View user profiles and their posts

## Constraints
- No private messaging
- No retweets/reposts
- No follower graph

## Tech Stack
- Backend: FastAPI (Python)
- Frontend: React
- Database: SQLite
- Auth: JWT
- Logging: Python logging
- Testing: Pytest, Jest
